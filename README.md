# Adhan

Adhan is a Rust developed application. It collects the prayer times from www.salahtimes.com/uk and outputs them to the console

## How to use

1. Clone repo
2. Build the binary
3. Run `adhan` to get the prayer times.
   - Run `adhan --help` to get a list of options
